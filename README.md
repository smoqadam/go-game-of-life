# Go Game-of-life

## How to use?

`go run main.go`


### Shortcuts

Start/Pause: `Ctrl+S`

New/Empty Screen: `Ctrl+N`

Random Cells: `Ctrl+R`

Active a cell: `Mouse Click`

Speed Up: `Ctrl+I`

Speed Down: `Ctrl+D`
